Device Preparation Guide

Follow these comprehensive steps to properly prepare your electronic devices for recycling or disposal.

Step 1: Backup Your Data

Why Backup?

Before you securely delete data from your device, make sure to backup all important information. Once securely deleted, data cannot be recovered.

Backup Options:
Cloud Storage
  • Google Drive, Dropbox, iCloud, OneDrive
  • Automatically syncs across devices
  • Typically requires subscription for large storage
External Hard Drive
  • One-time purchase, no subscription
  • Faster transfers for large amounts of data
  • Physical storage you control
Important: Verify your backup is complete and accessible before proceeding with data deletion.

Step 2: Device-Specific Preparation

Different devices require different preparation steps. Select your device type below for specific instructions:

Computer/Laptop Preparation

  1. Sign out of all accounts (email, cloud services, browsers)
  2. Deauthorize software that has limited installations
  3. Back up your data to an external drive or cloud service
  4. Use our secure deletion tool to wipe sensitive files
  5. For full disk wiping:
    • Windows 10/11: Settings > Update & Security > Recovery > Reset this PC > Remove everything
    • Select the option to "Remove files and clean the drive" for more thorough deletion
  6. For maximum security: Use third-party disk wiping tools like DBAN (Darik's Boot and Nuke) for HDDs
  7. For SSDs: Use manufacturer's secure erase utility or the built-in Windows reset with drive cleaning

  2. Deauthorize iTunes (if applicable): Account > Authorizations > Deauthorize This Computer

  4. Disable Find My Mac: System Preferences > Apple ID > iCloud > Find My Mac
  5. For macOS Monterey or newer:

  6. For older macOS versions:

    • Use Disk Utility to erase the drive (select secure erase options if available)
    • Reinstall macOS
  7. For T2 or M1/M2 Macs: The built-in erasure is already secure due to hardware encryption

  1. Back up important data to external storage
  2. For secure deletion:
    • Use the shred command for individual files: shred -vzn 3 filename
    • For entire drives (for HDDs): shred -vzn 3 /dev/sdX (replace X with your drive letter)
    • For SSDs, use hdparm with the secure erase command or the manufacturer's utilities
  3. Alternative tools: DBAN, secure-delete package (srm, sfill, etc.)
  4. For encrypted drives: Simply wiping the encryption keys is usually sufficient
Note: For computers with solid-state drives (SSDs), traditional secure deletion methods may not be as effective due to wear leveling. Consider using built-in secure erase functions or full-disk encryption before standard deletion.

Smartphone Preparation

  1. Back up important data using Google Backup or third-party apps
  2. Remove SIM card and external storage (SD card if applicable)
  3. Sign out of all accounts:
    • Settings > Accounts > Google (and other accounts) > Remove account
  4. Remove device from Google Find My Device:
    • Visit Find My Device on another device
    • Select your device and click "Remove device"
  5. Factory reset the device:
    • Settings > System > Reset options > Erase all data (factory reset)
    • Select "Erase all data" and confirm with your PIN/pattern
  6. After reset, verify that the device boots to the setup screen

Storage Device Preparation

  1. Back up any important data that you want to keep
  2. For basic deletion:
    • Connect the drive to a computer
    • Format the drive using your operating system's disk management tools
    • On Windows: Right-click the drive > Format > Check "Quick Format" > Start
    • On Mac: Disk Utility > Select drive > Erase > Choose format > Erase
  3. For secure deletion (HDDs):
    • Use our secure deletion tool for thorough data removal
    • Or use specialized software like DBAN, Eraser, or similar tools
  4. For external SSDs:
    • Use the manufacturer's secure erase utility if available
    • For encrypted drives, simply reset the encryption key

Step 3: Secure Data Deletion

Understanding Deletion Methods

When you normally delete a file, the operating system only removes the reference to the file, not the actual data. The data remains on the storage device until it's overwritten by new data.

To ensure your data cannot be recovered, you need to use secure deletion methods that overwrite the data multiple times.

Deletion Method Comparison
Method Security Level Speed Best For
Standard (1-pass) Basic Fast Non-sensitive data
DoD 5220.22-M (3-pass) Medium Moderate Personal/business data
Gutmann (35-pass) Very High Very Slow Highly sensitive data
Random (customizable) Variable Depends on passes Customized needs

Special Considerations
Solid-State Drives (SSDs) and Flash Storage

Traditional secure deletion methods are less effective on SSDs and flash storage due to:

  • Wear leveling: Redistributes writes to extend the drive's life
  • Over-provisioning: Reserved space not accessible through normal means
  • Garbage collection: Background process that may preserve data

For SSDs: Use the manufacturer's secure erase command, enable full-disk encryption before deletion, or use the built-in secure reset function in your operating system.

Step 4: Prepare for Recycling

Final Checklist Before Recycling

Physical Preparation

  • Batteries: If easily removable, remove batteries as they may need to be recycled separately. Many recycling centers require this.
  • Memory cards/SIM cards: Remove all memory cards, SIM cards, and other removable storage.
  • Packaging: If available, use original packaging for transport to prevent damage.
  • Labeling: Consider labeling the device as "data wiped" or "ready for recycling."
  • Documentation: Keep any certificates of secure deletion for your records.
